Build flow

`.env` values are generated before Angular runs.

`npm start` writes `src/environments/environment.ts` from your `.env` file, then starts `ng serve`.

`npm run build` regenerates the environment in production mode before creating the optimized bundle.

Only variables prefixed with `NG_APP_` are exposed to the browser app.
